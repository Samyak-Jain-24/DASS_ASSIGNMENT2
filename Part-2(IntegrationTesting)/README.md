# Part-2: Integration Testing — StreetRace Manager

This folder contains the implementation and integration testing artifacts for the StreetRace Manager CLI system.

## Modules
Required:
- Registration
- Crew Management
- Inventory
- Race Management
- Results
- Mission Planning

Additional:
- Maintenance (damage tracking and repair)
- Reputation (crew reputation tracking)

## Quick Start
```bash
python -m streetrace_manager.main
```
