Defects Found and Corrected (Error Log, 20 total):

Error #1
- Failing test(s): test_cards_remaining_empty_expected_zero
- Module: moneypoly/cards.py
- Root cause: CardDeck.cards_remaining() used modulo with len(self.cards) even when the deck was empty, causing ZeroDivisionError.
- Fix applied: Added an empty-deck guard in cards_remaining() that returns 0 when self.cards is empty.
- Commit message used: Error 1: Fix CardDeck empty-deck cards_remaining ZeroDivisionError

Error #2
- Failing test(s): test_roll_calls_randint_with_full_six_sided_range
- Module: moneypoly/dice.py
- Root cause: Dice.roll() used random.randint(1, 5) for both dice, excluding face value 6.
- Fix applied: Updated Dice.roll() to use random.randint(1, 6) for both die1 and die2.
- Commit message used: Error 2: Fix Dice.roll range to use 1-6 for both dice

Error #3
- Failing test(s): test_find_winner_should_pick_richest
- Module: moneypoly/game.py
- Root cause: Game.find_winner() used min(...) with net_worth, which selected the poorest player instead of the richest.
- Fix applied: Replaced min(...) with max(...) in find_winner() to return the player with highest net worth.
- Commit message used: Error 3: Fix find_winner to select highest net worth player

Error #4
- Failing test(s): test_player_move_passing_go_collects_salary

- Commit message used: Error 4: Fix Player.move to award GO salary when passing Go
